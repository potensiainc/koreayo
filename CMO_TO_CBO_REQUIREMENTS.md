### 1.1 수익 가정 검증

#### 가정 1: 프리미엄 멤버십 전환율 3%
```
근거:
- 업계 평균: 2-5% (SaaS 제품)
- Reddit Premium: ~1% (참고)
- Medium Membership: ~3-5%

우리 상황:
- 타겟: 한국 거주 외국인 (고관여 사용자)
- 가치 제안: 실용적 정보 (비자, 주거 등)
- 경쟁: 무료 대안 많음 (Reddit, Facebook)

CBO 질문:
1. 3% 전환율이 현실적인가?
2. 전환을 높이기 위한 추가 전략?
3. Freemium 가치 차별화 충분한가?
```

**리스크**: 전환율 1%일 경우 수익 66% 감소
**완화책**:
- 첫 달 50% 할인 (행동 유도)
- 연간 결제 할인 (₩99,000 → ₩79,000)
- 기업용 플랜 (B2B) 추가

## 5. 재무 계획

### 5.1 자금 조달 옵션

#### Option A: 자체 자금 (Bootstrapping)
```
장점:
- 지분 희석 없음
- 의사결정 자율성
- 빠른 실행

단점:
- 성장 속도 제한
- 위험 부담 높음

필요 자금: ₩60,000,000 (초기 6개월)
```

#### Option B: 엔젤 투자
```
장점:
- 빠른 성장 가능
- 멘토링/네트워크

단점:
- 지분 희석 (10-20%)
- 투자자 압박

목표 금액: ₩200,000,000
Valuation: ₩1,000,000,000 (Pre-money)
지분 제공: 20%
```

#### Option C: 정부 지원 (K-Startup, 서울시)
```
장점:
- 낮은 이자/무상 지원
- 지분 희석 없음

단점:
- 신청 조건 까다로움
- 승인 기간 길음 (3-6개월)

목표 금액: ₩50,000,000-100,000,000
```

#### Option D: 대출
```
장점:
- 빠른 자금 확보
- 지분 유지

단점:
- 이자 부담
- 상환 압박

금액: ₩100,000,000
이자율: 5%
상환 기간: 5년
```
